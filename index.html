02 — Projeto em Destaque

MotoJá

O projeto que mais representa onde estou e para onde quero ir.

MotoJá

Plataforma de Mobilidade

Plataforma de mobilidade urbana para mototáxi. Conecta passageiros a mototaxistas de forma rápida e organizada — pensada para funcionar de verdade, principalmente em cidades onde aplicativos convencionais não chegam.

Problema que resolve

  • Mototáxi é essencial em cidades pequenas e médias, mas não tem plataforma digital
  • Passageiros não têm como rastrear corridas ou escolher motoristas
  • Mototaxistas perdem corridas por falta de organização
  • Sem sistema, não há segurança nem registro das viagens

Principais funcionalidades

  • Cadastro e autenticação de passageiros e mototaxistas
  • Solicitação de corrida com localização em tempo real
  • Cálculo automático de tarifa por distância
  • Histórico de corridas e avaliação de motoristas
  • Painel administrativo com visão geral das operações
  • Notificações de status da corrida

Tecnologias do projeto

Node.js Express PostgreSQL REST API Git Docker HTML/CSS/JS

Próximos passos

  • Implementar geolocalização real com mapa interativo
  • Criar versão mobile com React Native
  • Testar em ambiente real com mototaxistas da região

03 — Plano de Carreira

Visão de 10 anos

Não é um roteiro fixo — é uma direção. Três fases pensadas para evoluir passo a passo.

ANOS 0–3
Base

Consolidar a base técnica e começar a trabalhar como desenvolvedor.

  • Concluir a graduação em ADS
  • Migrar para uma vaga de backend ou DevOps
  • Aprofundar em Node.js, Python, Docker e Cloud (AWS)
  • Tirar certificações: LPIC-1, AWS Cloud Practitioner
  • Construir portfólio com projetos reais (como o MotoJá)
  • Melhorar o inglês técnico
ANOS 3–5
Crescimento

Ganhar experiência com projetos maiores e aprofundar em arquitetura.

  • Trabalhar em projetos de média/alta complexidade
  • Estudar microsserviços, sistemas distribuídos e cloud avançado
  • Certificações: AWS Solutions Architect, Kubernetes (CKA)
  • Iniciar uma pós-graduação em Engenharia de Software
  • Começar um SaaS próprio como laboratório prático
ANOS 5–10
Estabilidade

Ter estabilidade profissional e, se fizer sentido, lançar um produto próprio.

  • Estar numa posição sólida como desenvolvedor ou arquiteto
  • Avaliar criação de produto ou startup com base no que aprendi
  • Estudar gestão de produto e visão de negócios
  • Ter independência financeira e profissional pela competência técnica
